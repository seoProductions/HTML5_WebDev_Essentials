The image tag <img>

the img tag is an empty tag. It only has attributes. Let's take a look at some

  • src: the relative URL of the image file
  • alt: detailed description of the image. (used by search engines and screen readers )
  • title: Displayed when image is hovered

here is a random photo I found lying around my desktop

Its good practice to add size specifications to images

this helps the browser load faster by knowing where to place the text, without needing the image to load!

you only need 2 attributes for this:

  • height (in pixels)
  • width (in pixels)

When you have an img tag inside a paragraph, you can align the image. Use the align attribute

  • align left or right of the screen

HTML5: Figure and Figcaption

Figure is a container which holds both img and figcaption. This makes it possible to associate an image with its caption. There was no way of doing this prior to HTML5

I like this way better than align. Figure can have multiple images: each sharing the same caption
