Table TAGS

Creating a table involves these HTML tags.

  • <table> : Container that holds the table
  • <tr> : (Table Row) Container that holds a row of data
  • <td> : (Table Data) Holds a cell of data
  • <th> : (Table Header) Same as Table Data - just more special.
    • Represents the header for a COLUMN or a ROW.
    • Good for Screen Readers & and CSS
    • Use it even if its empty - so that It renders correctly.
    • scope="" attribute indicated weather it is a column or row.
      ( row / col )

Good to have TAGS

These tags are helpfull when structuring a long table full of data. We seperate sections of the table into 3

  • <thead> : (Table Head) Container
  • <tbody> : (Table Body) Container
  • <tfoot> : (Table Footer) Container

We do this to better style and structure the table. We can even add good functionality
