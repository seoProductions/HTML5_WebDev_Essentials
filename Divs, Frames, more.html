Div & Span

These 2 are mostly for cosmetic purposes.

  • div is a block container. Always starts on a new line and extends for the width of the page

Meta-Deta

The meta tag lives inside the Header of the page. It uses attributes to describe usefull info about the page.

Now, the meta tag has a ton of possible attributes. lets look over them

  • name="": - the identifier for content. Used by the server
  • content="": - the info passed to the server.
  • http-equiv-"": - for the search engine's out there
Common Names

Some commonly used name's by convention are the following:

  • description - what the page is about. Dont exceed 155 char's
  • keywords - keywords the user can search for on google. not too practicle in todays web
  • robots - info to tell the search engine. Some values include:
    • noindex - dont add this page to the search results
    • nofollow - you can add this page, but not the links inside the page.
Common http-equivalents
  • author
  • pragma - prevents browser from storing cache data from the site.
  • expires - limit caching up to a certain date.
NOTE: The content property is included by all these attributes.
